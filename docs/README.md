# WM03_Ticket_System

## Overview

This project is a Student Support Ticket Management System built in Python with Tkinter and SQLite.  
It allows IT staff to add, search, filter, update, and report on student support tickets.

## Folder Structure

- app/ → Python source code files
- data/ → SQLite database (tickets.db)
- docs/ → User manual, technical report
- tests/ → Test plan and bug log
- screenshots/ → Evidence screenshots

## How to Run

1. Install Python 3.10+.
2. Run `pip install tk sqlite3` (if not already included).
3. Navigate to the project folder.
4. Run the application:
   ```bash
   python app/main.py
   ```
